## ⚡ 【核心神通】

此法宝内蕴四大秘境，分别对应**天时、地利、人和、玄机**，且听贫道细细道来：

### 1. 🌌 沉浸式入门仪式 · 盘古开天
*   **混沌初开**：道友初入此地，只见星空混沌，八卦旋转，仿佛重回宇宙未开之时。
*   **开天辟地**：只需轻点中心法阵，触发“天地分离”之异象，瞬间从混沌步入有序之境。此乃**仪式感**之大成，助道友静心凝神，准备接收天机。

### 2. 🧭 四大修炼秘境

#### 🕰️ 秘境一：天时（八卦罗盘）
> *“顺天者昌，逆天者亡。”*
*   **万年历法**：公历农历对照，每日宜忌一目了然，助道友把握时辰。
*   **每日运势**：基于当日卦象推演，详尽列出今日宜做何事，忌犯何煞。
*   **择吉日**：嫁娶、飞升（搬家）、开宗立派（开业），皆可选良辰吉日。

#### 🧭 秘境二：地利（风水罗盘）
> *“藏风聚气，得水为上。”*
*   **动态罗盘**：内置灵力罗盘，自动旋转，标示今日**喜神、财神、阳贵**方位，助道友寻宝纳福。
*   **出行指南**：结合今日冲煞，指引道友出行方向，避开凶险之地。

#### 🤝 秘境三：人和（通灵玉佩）
> *“人和则万事兴。”*
*   **八字算命**：输入生辰八字，自动推演四柱，分析五行能量分布，以此窥探命理根基。
*   **姓名测算**：依据笔画数理（三才五格），评测姓名吉凶，助道友改名换运。
*   **生肖运势**：详解十二生肖之年运、月运，知晓本命年之劫数与机遇。

#### 🔮 秘境四：玄机（天书卷轴）
> *“机不可失，时不再来。”*
*   **周公解梦**：梦中见何物？输入关键词，查阅周公天书，解析梦境寓意。
*   **六爻占卜**：模拟铜钱起卦，心诚则灵，预测事物发展吉凶。
*   **祈福许愿**：电子祈福法阵，寄托道友之宏愿，直达天听。

### 3. ✨ 视觉幻术
*   **五行流光**：背景随当日五行属性，流转不同色泽之灵气。
*   **粒子星空**：以 Canvas 绘制亿万星辰，闪烁频率随日期变幻，宛如置身银河。
*   **灵力反馈**：所有法阵（按钮）均带悬浮发光、缩放旋转之效，指尖触碰，灵力激荡。

## 🚀 【部署法门】

此法宝乃**纯静态**之物，无需后端服务器，无需搭建洞府（复杂环境），随处可安放。

### 方法一：本地参悟
1.  施展 git 法术克隆仓库：
    ```bash
    git clone https://github.com/your-username/yellow-calendar.git
    ```
2.  直接以神识（浏览器）打开 `index.html` 即可运行。

### 方法二：飞升上云（GitHub Pages）
1.  在 GitHub 开辟新洞府（Repository）。
2.  上传 `index.html` 法宝。
3.  进入设置，开启 **Pages** 功能，选择 `main` 分支。
4.  片刻之后，即可获得云端链接，供天下道友访问。

*(亦可部署于 Vercel、Netlify 等仙山福地)*
