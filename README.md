# xiuxian
# 太虚仙门 · 问道长生

> 一个沉浸式的中国修仙主题单页网站，融合道家玄学、养生理念与互动玩法。

## 简介

本项目是一个纯前端单页应用（SPA），旨在为"求道者"提供一个完整的修仙模拟体验。网站以中国古代修真体系为核心，详细阐述了从炼气到合道的境界划分、功法修炼、丹药炼制等核心内容，并创新性地将现代健康生活理念（如睡眠、饮食、运动）转化为修仙术语进行解读。

界面设计采用新中式玻璃拟态风格，配合动态星空、灵气粒子与飘动符文背景，营造出清静无为的道家氛围。

## 功能特性

### 1. 完整的境界体系
- **七大境界**：炼气、筑基、金丹、元婴、化神、返虚、合道。
- **五脏对应**：每个境界对应人体脏腑（肺、肾、心、肝、脾）与五行属性，点击可查看详细生理与玄学解释。
- **详细解析**：包含寿元、能力解锁、天劫考验、肉身变化等详细设定。

### 2. 功法秘籍库
- **分类明确**：基础、进阶、秘传、特殊法门（剑修、符修、阵修等）。
- **修炼周期**：每门功法包含详细的修炼天数、每日具体内容及预期效果（参考修仙小说设定）。
- **互动学习**：点击即可查看功法口诀与修炼要点，模拟真实修行过程。

### 3. 丹药宝典
- **灵丹妙药**：聚气丹、筑基丹、金丹丹等各级丹药详细介绍与功效说明。
- **天材地宝**：千年人参、龙血草、凤羽花等稀有灵材图鉴。
- **丹方秘录**：详细的炼制材料与火候控制方法。

### 4. 养生之道（玄学化解读）
- **子时养阴**：将睡眠与肾阴收藏结合，提供玄学与科学双重解释。
- **卯时采气**：晨练与阳气吸纳的关联。
- **饮食调和**：五味入五脏的五行膳食指南。

### 5. 神仙列传
- 彭祖、张三丰、吕洞宾、老子、葛洪、陈抟等历史/传说人物的生平、道法感悟与民间故事。

### 6. 互动修行中心
- **打坐冥想**：实时计时器，记录修行时长，增加灵力与进度。
- **灵根测试**：趣味测试，判断你的五行灵根属性与修炼资质。
- **每日任务**：模拟修行日常，追踪完成状态。

### 7. 试炼场（互动小游戏）
- **神识反应**：测试点击速度，评估神识敏锐度与反应等级。
- **定力试炼**：在干扰中保持专注，点击太极图标维持定力，干扰会越来越强。
- **记忆阵法**：类似 Simon 游戏，记忆符文闪烁顺序，考验神识记忆。
- **悟性问答**：道家经典知识问答，测试对大道的理解。

### 8. 视觉体验
- **动态背景**：星空闪烁、灵气粒子上升、漂浮符文。
- **玻璃拟态**：现代化的磨砂玻璃卡片设计，融合水墨山水意境。
- **响应式布局**：完美适配手机端与桌面端。

## 技术栈

- **HTML5**：语义化结构。
- **Tailwind CSS**：通过 CDN 引入，实现原子化 CSS 设计与自定义主题。
- **原生 JavaScript**：无框架依赖，轻量高效，包含复杂的状态管理。
- **Google Fonts**：使用 'Ma Shan Zheng'（马善政楷书）和 'Noto Serif SC'（思源宋体）营造古风氛围。
- **Font Awesome**：图标库。

## 如何使用

1.  **下载**：直接下载 `index.html` 文件。
2.  **运行**：双击文件在浏览器中打开即可，无需服务器环境。
3.  **部署**：可直接上传至 GitHub Pages、Vercel 或 Netlify。


## 设计理念

### 配色方案
- **墨黑 (#0a0e14)**：背景主色，象征混沌初开。
- **纸白 (#f5f0e6)**：文字主色，象征宣纸质感。
- **朱砂红 (#c53d43)**：警示与重点，道教符咒常用色。
- **玉青 (#5a7e7a)**：生机与灵气，象征木属性。
- **金 (#d4a853)**：尊贵与金丹，象征成就。

### 内容架构
- **"炼精化气"**：将现代健康习惯（早睡早起、运动）转化为修仙术语，增加趣味性与执行力。
- **"功法周期"**：参考《抱朴子》及网络修仙小说设定，为每门功法设定了具体的修炼天数和阶段目标，增强代入感。
