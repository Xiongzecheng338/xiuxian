# 太虚仙门 · 问道长生

一个融合经典仙侠与玄幻元素的完整修仙世界观设定与互动体验平台。

## 项目简介

本项目构建了一个完整的修仙世界，包含境界体系、功法秘籍、丹药宝典、养生之道、神仙列传等内容，并提供了打坐冥想、灵根测试、门派选择、秘境探索等互动功能。

## 技术栈

- **前端框架**：原生HTML5, CSS3, JavaScript
- **CSS框架**：Tailwind CSS
- **图标库**：Font Awesome
- **字体**：Google Fonts (Ma Shan Zheng, Noto Serif SC)
- **数据存储**：LocalStorage
- **项目结构**：模块化设计

## 快速开始

### 安装与运行

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd xiuxian
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动本地服务器**
   ```bash
   # 使用npm脚本
   npm start
   
   # 或直接使用http-server
   http-server -p 8000 -c-1
   ```

4. **访问项目**
   打开浏览器，访问 `http://localhost:8000`

### 开发指南

1. **修改数据**
   - 编辑 `src/data/realmData.js` 文件修改境界、门派等数据

2. **添加新功能**
   - 在 `src/modules/` 目录下创建新的功能模块
   - 在 `src/app.js` 中导入并注册新模块

3. **修改样式**
   - 在 `index.html` 的 `<style>` 标签中修改样式
   - 或在 `<head>` 中添加外部CSS文件

4. **构建与部署**
   ```bash
   # 构建项目
   npm run build
   
   # 部署到site目录
   npm run deploy
   ```

## 使用说明

### 基本操作

1. **导航**：点击顶部导航栏或移动端菜单切换页面
2. **语言切换**：点击右上角语言按钮切换中英文
3. **打坐冥想**：在修行中心页面点击"开始冥想"按钮
4. **灵根测试**：在修行中心页面点击"开始测试"按钮
5. **门派选择**：在修行中心页面选择一个门派加入
6. **秘境探索**：在修行中心页面点击"秘境探索"按钮
7. **法宝系统**：在修行中心页面点击"法宝系统"按钮
8. **传承系统**：在修行中心页面点击"传承系统"按钮

### 修炼系统

1. **打坐冥想**：通过打坐积累修为，达到一定修为后自动突破境界
2. **境界突破**：突破境界后获得新的能力和寿元
3. **奇遇系统**：突破境界时有几率触发奇遇事件
4. **瓶颈系统**：每个境界都有突破瓶颈，需要完成特定任务

### 特色功能

1. **每日签道**：随机生成修真语录
2. **修真运势**：每日显示适宜的修炼活动
3. **悟性问答**：解答道家难题，测试悟性
4. **记忆游戏**：测试记忆力，锻炼神识
5. **定力试炼**：测试专注力，锻炼定力
6. **反应测试**：测试反应速度，锻炼神识

## 数据持久化
